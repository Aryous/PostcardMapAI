# UI Audit — MapPostcard AI
> 日期：2026-03-17 | 分支：feature/bold-ui-redesign

---

## 总体结论

英雄页已建立清晰的"制图学·编辑杂志"视觉语言，主应用没有跟上。
从着陆页进入主应用，像从精品旅行杂志封面切换进了 shadcn 后台管理系统。

---

## 任务清单（按优先级）

### 🔴 严重 (S)

- [ ] **S1** — `LandingPage` How It Works 第一步描述了不存在的功能
  "用矩形框选城市街区" — 实际流程是移动地图自动触发，无手动框选。
  修复：改第一步描述为"移动/缩放地图到目标区域"。
  文件：`LandingPage.tsx:365`, `translations.ts`
  技能：`/clarify`

- [ ] **S2** — `App.tsx` 主容器仍用 `bg-slate-50`（冷蓝灰）
  主应用根容器颜色未随品牌配色更新，与面板奶油色形成色差。
  修复：`bg-slate-50` → `bg-[#ede7d5]`
  文件：`App.tsx:201`
  技能：`/colorize`

---

### 🟠 高优先级 (H)

- [ ] **H1** — 设计语言断层：英雄页和主应用像两个产品
  控制面板的结构语言（segmented control、pill tags、dashed upload）是通用 SaaS，
  与英雄页的制图仪器/旅行杂志风格毫无连接。
  修复：用 `overdrive` 重新设计控制面板，与英雄页风格统一。
  文件：`ControlPanel.tsx`
  技能：`/overdrive`

- [ ] **H2** — 画廊区是标准 "identical card grid" 反模式
  3×2 等宽等高卡片，无节奏变化，无重点突出。
  修复：打破等宽布局，用杂志式排版（大图配小图，错位）。
  文件：`LandingPage.tsx:340`
  技能：`/arrange`

- [ ] **H3** — 控制面板头部按钮触控目标过小（约 28px，低于 44px 标准）
  历史按钮、语言切换、开发者按钮均不符合移动端最小触控目标。
  修复：`p-1.5` → `p-2.5` 或添加 `min-w-[44px] min-h-[44px]`
  文件：`ControlPanel.tsx:181-207`
  技能：`/adapt`

- [ ] **H4** — 语言切换用 emoji 国旗，偏离精致品牌基调
  `🇺🇸 | 🇨🇳` 在精品制图风格里格格不入，跨系统渲染不一致，无障碍性差。
  修复：改为 `EN · 中` DM Mono 文字 + `aria-label`
  文件：`ControlPanel.tsx:203-207`
  技能：`/clarify`

---

### 🟡 中优先级 (M)

- [ ] **M1** — 所有自定义交互元素缺乏 focus-visible 指示器
  Tailwind CDN 会移除默认 outline，键盘用户无法感知焦点，违反 WCAG 2.1 AA SC 2.4.7。
  文件：`ControlPanel.tsx`, `LandingPage.tsx` 所有按钮
  技能：`/harden`

- [ ] **M2** — 画廊移动端 3 列布局会溢出
  `gridTemplateColumns: 'repeat(3, 1fr)'` 内联样式无响应式覆盖。
  修复：添加 CSS 媒体查询，移动端改为 1-2 列。
  文件：`LandingPage.tsx:341`, `LandingPage.css`
  技能：`/adapt`

- [ ] **M3** — How It Works 英文副标题字号小于正文（层次逻辑混乱）
  英文 13px 出现在正文 14px 之前，阅读节奏被打断。
  文件：`LandingPage.tsx:372`
  技能：`/typeset`

- [ ] **M4** — 控制面板缺乏任何制图/旅行风格锚点
  即使配色统一，结构语言仍是通用 SaaS。（与 H1 联动，H1 修复后此项自然解决）
  文件：`ControlPanel.tsx`

---

### 🟢 低优先级 (L)

- [ ] **L1** — 画廊图片无 `loading="lazy"` 属性
  6 张 `/gallery/*.png` 首屏外仍会立即下载。
  文件：`LandingPage.tsx:134-140`

- [ ] **L2** — 历史记录按钮只有 `title` 无 `aria-label`
  `title` 在移动端不显示，辅助技术行为不一致。
  文件：`ControlPanel.tsx:190`

- [ ] **L3** — CTA 区域硬编码了线上域名
  `postcard-map-ai.vercel.app` 写死在组件里，换域名需改代码。
  文件：`LandingPage.tsx:402`

## 正面发现（保留）

- ViewfinderOverlay 弹性角标 + 边框扫描迹线 + 扫描光束 — 整个产品最有辨识度的元素
- 英雄页 SVG 制图背景 + 渐变遮罩 + 左栏编辑排版 — 视觉语言清晰
- Forest `#2a4535` 品牌色在两个界面保持一致
- Compass 弹性寻北动画 — 有趣的细节

---

## 推荐执行顺序

```
S2 (1行修复) → S1 (文案修复) → H4 (语言切换) → H3 (触控目标)
→ H1+M4 (控制面板 overdrive，最大工作量)
→ H2+M2 (画廊重排 + 响应式)
→ M1 (focus indicators)
→ L1/L2/L3 (收尾)
```
